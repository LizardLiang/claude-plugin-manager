# Development Phases

This folder contains the phased breakdown of the Claude Code Plugin Manager development plan.

## Phase Overview

| Phase | Version | Focus | Status |
|-------|---------|-------|--------|
| [Phase 1](phase-1-mvp.md) | v0.1 | MVP - Basic browsing, search, global install | Not Started |
| [Phase 2](phase-2-core-features.md) | v0.5 | Core - Projects, multiple marketplaces, groups | Not Started |
| [Phase 3](phase-3-full-release.md) | v1.0 | Full Release - Updates, config editor, polish | Not Started |
| [Phase 4](phase-4-future.md) | v2.0+ | Future - Cross-platform, CLI, analytics | Not Started |

## Dependency Chain

```
Phase 1 (MVP)
    ↓
Phase 2 (Core Features)
    ↓
Phase 3 (Full Release)
    ↓
Phase 4 (Future)
```

## Key Milestones

- **v0.1**: Users can browse and install plugins globally from default marketplace
- **v0.5**: Full plugin management with projects, groups, and multiple marketplaces
- **v1.0**: Production-ready with updates, configuration, and polished UX
- **v2.0**: Cross-platform support and CLI companion tool
